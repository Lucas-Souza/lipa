PROGRAMAS
------------
 1 - Pares mais próximos
       - Pasta:   1
       - Arquivo: 1.c
     O programa inicialmente solicita um valor n de entradas. Digite um valor inteiro maior ou igual a 2 e aperte enter.
     Caso o valor seja menor que 2, ele continuará solicitando o número n de entradas.
     Após isso, o programa solicitará as entradas. Digite um valor e aperte enter. Repita o processo n-1 vezes
     O par com a menor diferença será informado após a última entrada de dados
     Lembre-se: utilize apenas números inteiros.

------------
 2 - Elemento único
       - Pasta:   2
       - Arquivo: 2.c
     O programa inicialmente solicita um valor n de entradas. Digite um valor inteiro maior ou igual a 2 e aperte enter.
     Após isso, o programa solicitará as entradas. Digite um valor e aperte enter. Repita o processo n-1 vezes
     Os elementos únicos no vetor serão impressos após a última solicitação.
     Lembre-se: utilize apenas números inteiros.

------------
 3 - Distribuição de frequência
       - Pasta:   3
       - Arquivo: 3.c
     O programa inicialmente solicita um valor n de entradas. Digite um valor inteiro maior ou igual a 2 e aperte enter.
     Após isso, o programa solicitará as entradas. Digite um valor e aperte enter. Repita o processo n-1 vezes.
     Os elementos que apareceram com a maior frequência serão impressos após a última entrada de dados.
     Lembre-se: utilize apenas números inteiros.
------------------
 4 - Fecho convexo
       - Pasta:   4
       - Arquivo: 4.c
     O programa solicita um número n de pares (coordenadas x e y do plano cartesiano). Digite um valor inteiro maior ou igual a 3 e aperte enter.
     Digite um par inteiro da seguinte forma: "x y". Aperte enter. Repita o processo n-1 vezes
     Após todas as entradas, o programa imprimirá todos os vértices do fecho convexo
     Lembre-se: utilize apenas números inteiros.
